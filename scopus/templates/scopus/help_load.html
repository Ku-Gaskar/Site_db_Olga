Для обновление таблиц статей scopus базы данных:
  1. Подготовить каталог на компьютере для скачивания файлов из "scopus.com"
  2. В системе "scopus.com" выполнить поиск статей по профилю университета
  3. Выбрать поля для экспорта в "BibTex" формате во вкладке "Экспортировать настройки документа" :
    • Автор(ы)
    • Идентификатор автора(ов)
    • Название документа
    • Год
    • EID
    • Название источника
    • том, выпуск, страницы
    • Количество цитирований
    • Источник и тип документа
    • DOI
    • Издатель
  4. Выполнить экспорт статей в формате "BibTex". Если статей больше чем 2000, то выполнить экспорт несколькими файлами с ипользованием фильтра (например: по годам)
  5. После завершения экспорта статей в "BibTex" формате выполнить экспорт статей в csv формате по тому же алгоритму
  6. Выбрать поля для экспорта в "csv" формате во вкладке "Экспортировать настройки документа" ВАЖНО!!!
    • Идентификатор автора(ов)
    • EID
  7. Когда файлы "*.bib" и "*.csv" будут готовы, нажмите "Готово" и в диалоговом окне выберите все эти файлы
  8. Процесс обновления будет запущен
