Для обновление таблиц статей wos базы данных:
  1. Подготовить каталог на компьютере для скачивания файлов из "wos.com"
  2. В системе "wos.com" выполнить поиск статей по профилю университета
  3. Выбрать поля для экспорта в "BibTex" формате во вкладке "My custom export selections (Web of Science Core Collection)" :
    • Author(s)
    • Title
    • Source
    • Times Cited Count
    • Accession Number
    • Authors Identifiers
    • ISSN
    • Document Type
    • Publisher Information
    • Page Count
    • Source Abbrev.
    • IDS Number
  4. Выполнить экспорт статей в формате "BibTex". Если статей больше чем 1000, то выполнить экспорт несколькими файлами с ипользованием "Records from: 1 to 1000"
  5. Когда файлы "*.bib" будут готовы, нажмите "Готово" и в диалоговом окне выберите все эти файлы
  6. Процесс обновления будет запущен
